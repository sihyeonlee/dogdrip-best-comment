# dogdrip-best-comment

## About
![Dogdrip-best-comment](/source/dogdrip-comment-capture.PNG)
개드립 커뮤니티 내에서 사용자가 설정한 페이지 내의 개드립 글 중, 설정한 좋아요 수 이상을 받은 댓글을 크롤링 합니다.

## How-to-use
1. 크롤링 시작할 페이지와 끝 페이지를 입력해줍니다.
2. 몇 개 이상의 좋아요를 받은 댓글을 필터링할지 입력해줍니다.
3. 검색 버튼을 누르면 크롤링을 시작합니다.
4. 하단에 표시된 상태 메시지를 통해서 검색 중인지 또한 어느 페이지 / 어느 게시물을 진행 중인지 확인할 수 있습니다.
5. 검색 중지 버튼을 누르면 모든 작업이 중지됩니다.
6. 초기화를 누르면 검색된 내용이 모두 제거됩니다.
7. 게시물 링크를 더블클릭하면 자동으로 해당 댓글 페이지가 열립니다.
8. 악용 방지를 위해서 한 번에 크롤링 가능한 페이지는 100페이지 이내입니다.
